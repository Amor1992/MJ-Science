# MJ-Science 实现计划

## 📋 项目概述

基于 PhET Interactive Simulations 设计理念，为 MJ 的两个孩子（一、二年级）创建互动式科学实验学习平台。

## 🎯 核心目标

1. **学习性** - 符合一二年级认知水平
2. **趣味性** - 游戏化设计，收集星星奖励
3. **交互性** - 拖拽、点击、滑动等直观操作
4. **可视化** - 让抽象概念变得可见可触

## 📚 实验列表（8个）

### 🔥 第一批（已完成：2/4）

#### ✅ 1. 平衡跷跷板 (balancing-act.html)
- **文件位置**: `experiments/balancing-act.html`
- **状态**: ✅ 已完成
- **核心功能**:
  - 拖拽不同质量物体（1kg, 2kg, 3kg）
  - 放在跷跷板不同位置
  - 实时计算力矩并显示倾斜
  - 平衡检测和星星奖励
  - 平滑动画过渡
- **教育目标**: 力矩、平衡、杠杆原理
- **适合年龄**: 6-9岁
- **难度**: ⭐ 简单

#### ⏳ 2. 面积建造者 (area-builder.html)
- **计划完成**: 第2-3天
- **核心功能**:
  - 网格画布，用方块建造形状
  - 实时计算面积和周长
  - 挑战模式：给定面积/周长要求
  - 比较两个形状的差异
- **教育目标**: 面积、周长、几何概念
- **适合年龄**: 6-9岁
- **难度**: ⭐ 简单

#### ⏳ 3. 分数配对 (fraction-matcher.html)
- **计划完成**: 第4-5天
- **核心功能**:
  - 左右两个托盘
  - 拖拽分数图形或数字配对
  - 8个难度等级
  - 等价分数验证（1/2 = 2/4 = 3/6）
  - 星星奖励系统
- **教育目标**: 分数、等价分数、数感
- **适合年龄**: 6-9岁
- **难度**: ⭐ 简单

### 💪 第二批（计划中）

#### ✅ 4. 力与运动基础 (forces-motion.html)
- **文件位置**: `physics/forces-motion.html`
- **状态**: ✅ 已完成
- **核心功能**:
  - 场景1: 拔河比赛
    - 调整双方人数和推力
    - 实时显示速度表、力度表
    - 绳子物理模拟（张力、阻尼）
  - 场景2: 推箱子
    - 施加推力
    - 调节摩擦力
    - 观察不同表面效果（冰面、木板、沙地、地毯、橡胶）
    - 实时显示净力、加速度、速度
- **教育目标**: 力、运动、摩擦力、惯性
- **适合年龄**: 7-9岁
- **难度**: ⭐⭐ 中等

#### ⏳ 5. 建造分数 (build-fraction.html)
- **计划完成**: 第9-10天
- **核心功能**:
  - 给定目标分数（如 3/4）
  - 用图形块拼凑出对应值
  - 多种模型：圆形、长方形、数轴
  - 从"部分"构建"整体"
- **教育目标**: 分数构建、逆向思维
- **适合年龄**: 6-8岁
- **难度**: ⭐⭐ 中等

#### ✅ 6. 物质状态基础 (states-of-matter.html)
- **文件位置**: `matter/states-of-matter.html`
- **状态**: ✅ 已完成
- **核心功能**:
  - 三种物质：水、氖、氧（不同的熔点和沸点）
  - 温度滑块（-200°C 到 200°C）
  - 宏观视角：观察物质整体外观变化（方形、圆形、气泡）
  - 微观视角：观察分子运动和排列（固态振动、液态流动、气态逃逸）
  - 相变动画和音效
  - 实时信息面板（熔点、沸点、当前状态、分子速度）
- **教育目标**: 固态/液态/气态、温度与分子运动、相变
- **适合年龄**: 7-9岁
- **难度**: ⭐⭐ 中等

## 💻 技术架构

### 前端技术栈
```
HTML5 + CSS3 + Vanilla JavaScript
├── 渲染：Canvas 2D API
├── 交互：原生拖拽 + 触摸事件
├── 动画：requestAnimationFrame
├── 布局：Flexbox + Grid
└── 存储：localStorage（进度保存）
```

### 项目结构
```
MJ-Science/
├── index.html                          # 首页导航
├── experiments/
│   ├── balancing-act.html             # ✅ 平衡跷跷板
│   ├── area-builder.html              # ⏳ 面积建造者
│   ├── fraction-matcher.html          # ⏳ 分数配对
│   ├── forces-motion.html             # ⏳ 力与运动
│   ├── build-fraction.html            # ⏳ 建造分数
│   ├── states-of-matter.html          # ⏳ 物质状态
│   ├── balloons-static.html           # ⏳ 气球静电
│   └── fractions-intro.html           # ⏳ 分数入门
├── shared/
│   ├── physics-engine.js              # 物理引擎（可选）
│   ├── ui-utils.js                    # UI 工具函数
│   └── animations.js                  # 动画库（可选）
├── assets/
│   ├── sounds/                        # 音效（可选）
│   └── images/                        # 图片资源（可选）
└── README.md                          # 使用说明
```

### 核心设计模式

#### 1. 统一的实验模板
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>实验名称</title>
    <style>
        /* 统一视觉风格 */
    </style>
</head>
<body>
    <!-- 导航栏 -->
    <!-- 游戏区域 -->
    <!-- 控制面板 -->
    <!-- 提示/帮助 -->

    <script>
        // 初始化
        function init() { }

        // 主循环
        function gameLoop() {
            update();
            draw();
            requestAnimationFrame(gameLoop);
        }

        // 交互事件
        canvas.addEventListener('mousedown', ...);
        canvas.addEventListener('touchstart', ...);
    </script>
</body>
</html>
```

#### 2. 星级奖励系统
- 每个实验有 3 颗星可收集
- 总星数显示在首页
- 使用 localStorage 保存进度

#### 3. 难度分级
- ⭐ 简单：6-7岁，基础概念
- ⭐⭐ 中等：7-8岁，需要思考
- ⭐⭐⭐ 困难：8-9岁，挑战性

#### 4. 响应式设计
- 支持桌面和移动设备
- 触摸事件兼容
- 自适应画布大小

## 🎨 设计规范

### 视觉风格
- **字体**: Comic Sans MS（亲切、儿童友好）
- **配色**: 渐变紫背景 + 白色卡片
- **图标**: Emoji 图标（直观、生动）
- **圆角**: 20px 圆角（柔和、友好）

### 交互设计
- **拖拽**: 流畅、视觉反馈
- **点击**: 明确的选中状态
- **反馈**: 即时响应 + 动画过渡
- **奖励**: 星星收集 + 庆祝动画

### 可访问性
- **触摸支持**: iPad、手机可用
- **键盘支持**: 基础功能可用键盘
- **文字大小**: 最小 16px
- **对比度**: 符合 WCAG 标准
